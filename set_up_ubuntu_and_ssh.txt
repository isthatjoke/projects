1. запуск терминала
2. установка обновлений установленных программ и библиотек: 
a) sudo apt update - получаем список программ и библиотек для обновления
б) sudo apt upgrade , далее при вопросе нажимаем Y, ввод и ждем конца установки
3. установка расширения для модификации ядра системы: sudo apt install gcc make perl
4. установка сервера ssh: sudo apt install openssh-server и после установки запустить сервер командой sudo service ssh start
5. командой ssh-keygen запустить генератор ssh ключей, а)согласиться с местой расположения ключей б)добавить или оставить пустой проверочное слово в)подтвердить проверочное слово
6. в полученном ответе генератора ключей пройти по адресу расположения файлов и прочитать содержимое файла id_rsa , cat ~/.ssh/id_rsa
7. скопировать код из терминала и сохранить на раб столе хостовой системы (в данном случае windows)
8. скопировать данные ключей ssh в файл authorized_keys командой  cat ~/.ssh/id_rsa >> ~/.ssh/authorized_keys  , cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
9. для последующего удобства просмотра и правки файлов установить MC командой sudo apt install mc
10. далее поправить конфиг ssh сервера в папке ~/etc/ssh/ssh_config и sshd_config , открыв их с помощью файлового менеджера mc командой sudo mc , далее выбрать необходимый файл для правки : раскомментировать PasswordAuthentication no, IdentityFile ~/.ssh/id_rsa, Port 22, PubkeyAuthentication yes
11. посмотреть команды для файерволла - ufw -help
12. добавить порт 22 в список разрешенных (требуется для доступа по ssh) командой sudo ufw allow 22, узнать ip ubuntu командой ip address
13. перезагрузить сервер ssh командой sudo service ssh restart
14. на виндоус создать файл ключ для PuTTy, указав ему текстовый файл с копированным ранее ключом ssh из ubuntu
15. далее на виндоус в программе PuTTy настроить соединение, добавив логин, путь к сгенерированному файлу ключа и указать ip addres ubuntu
16. подключиться к ubuntu с терминала хост системы и работать
